# Examples

These examples are executable TypeScript lessons that reuse the ReBAC domain
ideas without being part of the server.

```text
examples/concurrency/     Promise concurrency and bounded parallelism
examples/generics/        Result<T, E> discriminated unions
examples/middleware/      Higher-order handlers and middleware composition
examples/authzhttp/       A tiny HTTP authorization boundary
```

Run them with the normal test suite:

```bash
npm test
```
